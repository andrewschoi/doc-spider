Doc Spider

Upload a zip of your source code and write GPT-generated documentation on nested functions

Approach:

Accept a zip file via a Express and Multer and sandbox the unzipped application via Docker

Perform simple DFS on folder hierarchy and target files with .js extension

Access function headers, arguments, and body via AST

Generate function documentation via requests to OpenAI

Limitations

Code support for other languages, i.e. C++ (AST properties are different)

Areas for improvement

It is usually the case that functions use other functions within their definitions, efficiently gathering context about other functions before writing documentation on a dependent function can improve documentation quality
